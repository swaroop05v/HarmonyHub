# HarmonyHub - Music Playlist Manager

Full-stack project (React frontend + Node/Express backend + MongoDB).

## Structure
- frontend/      -> React app (src, public)
- backend/       -> Node + Express server, Mongoose models

## Setup

### Backend
1. cd backend
2. npm install
3. Create a `.env` file based on `.env.sample`
4. npm run dev

### Frontend
1. cd frontend
2. npm install
3. npm start

## Notes
This is a minimal skeleton intended for a college project demo. Expand UI and add validations/auth hardening before production.
